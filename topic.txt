Certainly! Here is the combined command set for both methods to install Python 3.6 using Homebrew and `pyenv`.

### Method 1: Install Python 3.6 using Homebrew directly

```bash
brew update
brew tap homebrew/cask-versions
brew install python@3.6
echo 'export PATH="/usr/local/opt/python@3.6/bin:$PATH"' >> ~/.bash_profile
source ~/.bash_profile
python3.6 --version
```

### Method 2: Install Python 3.6 using `pyenv`

```bash
brew update
brew install pyenv
brew install openssl readline sqlite3 xz zlib
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bash_profile
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bash_profile
echo 'eval "$(pyenv init --path)"' >> ~/.bash_profile
source ~/.bash_profile
pyenv install 3.6.15
pyenv global 3.6.15
python --version
```

You can copy and paste the commands above into your terminal depending on which method you prefer to use. If you encounter any issues, please provide specific error messages so I can assist you further.
